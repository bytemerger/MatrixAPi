# MatrixAPI
##### The API multiplies two matrices

## Endpoint
```post /api/multiply``` 
requires two matrices namely matrixA and matrixB
matrix: Two Dimensional array with representing the rows and column eg ```[
    [32, 54, 65],
    [45, 54, 65],
    [45, 32, 65],
    [45, 32, 54]
];```

## To Test
To test just
```run php artisan test```
